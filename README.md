## 🚀 Quick Start

### Option 1: Use Online
Just visit the live link and start creating tasks!

### Option 2: Run Locally

1. **Clone or Download**
   ```bash
   git clone https://github.com/YOUR_USERNAME/todo-list.git
   cd todo-list
   ```

2. **Run a Local Server**
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Or using Node.js
   npx http-server
   
   # Or open in VS Code and use Live Server extension
   ```

3. **Open in Browser**
   ```
   http://localhost:8000
   ```

## 📖 How to Use

### Getting Started
1. **Sign Up** - Create your account with email and password
2. **Log In** - Access your personal task list
3. **Add Task** - Type a task and click the "+" button
4. **Manage Tasks** - Edit, complete, or delete tasks as needed
5. **Log Out** - Securely sign out when done

### Task Actions
- **Click Status Button** - Mark task as pending/completed
- **Click Edit Button** - Change task description
- **Click Delete Button** - Remove task permanently
- **Double Click Task** - Quick edit (if enabled)

## 🐛 Troubleshooting

### Can't see tasks after login?
- Refresh the page (F5 or Cmd+R)
- Clear browser cache
- Check if you're logged in correctly

### Tasks disappeared?
- Log out and log back in
- Check if you deleted them by mistake
- Contact support if issue persists

### Button not working?
- Try refreshing the page
- Clear browser cache
- Check browser console (F12) for errors

### Password reset not working?
- Check spam folder for reset email
- Make sure you entered correct email
- Wait a few minutes and try again
